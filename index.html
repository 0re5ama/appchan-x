The latest stable releases of Chrome, Firefox, and Opera are supported.

Install Appchan X

Chrome:
  1. Install Appchan X. That's it.
  2. Appchan X will be automatically updated by Chrome.
Firefox:
  1. If you don't have Greasemonkey installed:
    1. Install Greasemonkey.
    2. Restart Firefox.
  2. Install Appchan X.
Opera:
  1. If you don't have Download Chrome Extension installed:
    1. Install Download Chrome Extension.
  2. Install Appchan X.
  3. Appchan X will be automatically disabled by Opera because it's considered a "Third Party Extension". Open the Extensions page and manually enable it.
